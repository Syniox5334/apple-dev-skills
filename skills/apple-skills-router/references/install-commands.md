# Install Commands

## Primary Command

`npx skills add gaelic-ghost/apple-dev-skills --skill <skill-name>`

Use this when the target skill is known and missing.

## Secondary Convenience Command

`npx skills add gaelic-ghost/apple-dev-skills --all`

Use this only when the user explicitly wants the whole repository installed.

## Common Starting Points

- Xcode workflow:
  `npx skills add gaelic-ghost/apple-dev-skills --skill apple-xcode-workflow`
- Dash docs:
  `npx skills add gaelic-ghost/apple-dev-skills --skill apple-dash-docsets`
- Swift package tools:
  `npx skills add gaelic-ghost/apple-dev-skills --skill apple-swift-package-bootstrap`
